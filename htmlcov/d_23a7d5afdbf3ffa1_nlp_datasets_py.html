30class HuggingfaceTextDataModule(RenateDataModule): 

31 """Data module wrapping Huggingface text datasets. 

32 

33 This is a convenience wrapper to expose a hugginface dataset as a `RenateDataModule`. Datasets 

34 will be pre-tokenized and will return `input, target = dataset[i]`, where `input` is a 

35 dictionary with fields `["input_ids", "attention_mask"]`, and `target` is a tensor. 

36 

37 We expect the dataset to have a "train" and a "test" split. An additional "validation" split 

38 will be used if present. Otherwise, a validation set may be split off of the training data 

39 using the `val_size` argument. 

40 

41 Args: 

42 data_path: the path to the folder containing the dataset files. 

43 dataset_name: Name of the dataset, see https://huggingface.co/datasets. This is a wrapper 

44 for text datasets only. 

45 input_column: Name of the column containing the input text. 

46 target_column: Name of the column containing the target (e.g., class label). 

47 tokenizer: Tokenizer to apply to the dataset. See https://huggingface.co/docs/tokenizers/ 

48 for more information on tokenizers. 

49 tokenize_kwargs: Keyword arguments to be passed to the tokenizer. Typical options are 

50 `max_length`, `padding` and `truncation`. See https://huggingface.co/docs/tokenizers/ 

51 for more information on tokenizers. If `None` is passed, this defaults to 

52 `{"padding": "max_length", max_length: 128, truncation: True}`. 

53 val_size: Fraction of the training data to be used for validation. 

54 seed: Seed used to fix random number generation. 

55 """
